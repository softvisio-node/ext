# INSTALL

## NPM
```
npm i ssh://git@bitbucket.org/softvisio/softvisio-ext
```

# YARN
```
# TODO not work for yarn v2
yarn add ssh://git@bitbucket.org/softvisio/softvisio-ext
```

# USE

## LATEST VERSION
```
import( "@softvisio/ext" );
```

## SPECIFIC VERSION
```
import( "@softvisio/ext/extjs-v7.1.0.js" );
```
